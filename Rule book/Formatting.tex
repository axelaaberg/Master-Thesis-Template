Language: English (US)
Oxford comma: Yes
Max depth subsections: 3 

Thousands separator: Space ( ). 
Decimal sign: Period (.). 
> Use \num{1234.56} in-text and in tables to comply with the above. 
> Use $$math$$ for math expressions.

Naming conventions;
> Headers (all levels): Title Case (https://titlecaseconverter.com/)
> Paragraphs: Sentence case.
> File names: Sentence case. Enumerate before if text file. Enumerate after if image. 
> Label names: lowercase. "fig:" OR "tab:" OR "eq:" OR "sec: AND description AND enumerate, separating description and number with underscore (\_).
> Captions: Sentence case
> Global variables: camelCase
> Unit declaration: Brackets []

Referencing;
> Reference names: lowercase. First author last name AND description, separated by an underscore. 
> textcite[page]{} in text. parencite[page]{} at the end of sentence.
> Reference location: place reference after the first sentence that uses that source. Then, this source applies until the next page range, next source, OR the next paragraph.
